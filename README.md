## 📡 Endpoints - User

### 🔹 Criar usuário
POST /user

### 🔹 Buscar usuário por ID
GET /user/{id}

### 🔹 Atualizar usuário
PUT /user/{id}

### 🔹 Deletar usuário
DELETE /user/{id}
